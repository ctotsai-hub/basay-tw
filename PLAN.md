# 凱達格蘭（巴賽語）～從記憶到再生～ 網站企劃書

> Domain: **basay.tw** ｜ Hosting: **GitHub Pages** ｜ Stack: **Static HTML / CSS / JS**

---

## 1. 專案目標（Purpose）

凱達格蘭族（Ketagalan）巴賽語（Basay）是分布於台灣北海岸、淡水河口與基隆一帶的平埔族語言，長期被視為「沉默的語言」。本網站以「**從記憶到再生**」為核心概念，致力於：

1. **保存**：彙整既有文獻、辭彙、語法紀錄，並以可檢索的形式公開。
2. **教育**：透過每日短句、會話集，讓學習者能每天接觸到活的巴賽語。
3. **研究**：為語言學者、人類學者提供研究成果的發表與交流平台。
4. **再生**：鼓勵族人與一般民眾使用巴賽語，讓語言重回生活。

## 2. 目標對象（Audience）

| Persona | 需求 | 進入點 |
|---|---|---|
| 族人與後裔 | 尋找祖先語言、學習日常會話 | 「今日的巴賽語」「會話集」|
| 語言愛好者・學生 | 想系統性學習語法 | 「文法基本」|
| 語言學・人類學研究者 | 查閱文獻、引用詞條 | 「研究成果」「辭典」|
| 一般民眾 | 好奇凱達格蘭是誰、想在淡水景點認字 | 首頁介紹、辭典 |

## 3. 資訊架構（Information Architecture）

```
basay.tw
├── /                    首頁（品牌・今日巴賽語・入口）
├── /grammar/            文法基本（音韻・語序・代名詞・動詞等）
├── /education/          教育推進
│   ├── /education/phrases/   會話集
│   └── /education/daily/     本日的巴賽語・新聞翻譯
├── /research/           論文與研究成果
├── /dictionary/         辭典檢索（Basay ↔ 中 / 日 / 英）
└── /about/              關於本計畫（可日後擴充）
```

## 4. 視覺方向（Visual Direction）

**主題**：淡水河口 → 八里海岸 → 大屯山系的色調。

| 角色 | 顏色 | HEX |
|---|---|---|
| 深海藍（夜的淡水河） | 基礎 | `#1a3a52` |
| 砂岸色（八里沙岸） | 背景 | `#f0e6d2` |
| 海藻綠（潮間帶） | 輔色 | `#5a7a6b` |
| 火山磚紅（大屯火山・祖先陶土） | 強調 | `#c86d4a` |
| 流木灰 | 文字副色 | `#6b5d54` |
| 墨黑 | 文字主色 | `#2c2620` |

字體：繁中使用 `"Noto Serif TC", "Source Han Serif"` 為主（典雅、適合研究內容）；羅馬字用 `"Inter", sans-serif`。

## 5. 技術選擇（Tech Choices）

- **Static HTML + Vanilla JS**：零建置、部署透明、長期可維護。  
- **辭典雙層架構**：
  - **主資料庫 → Notion**（編輯友善、非技術研究者也能直接維護）。
  - **快取 → `data/dictionary.json`**（從 Notion 匯出 / 同步，供 basay.tw 前端快速檢索）。
  - 使用者在 `/dictionary/` 頁既可即時檢索本地快取，也可一鍵跳轉 Notion 原始資料庫。
- **GitHub Pages + CNAME**：直接將 main branch 指向 `basay.tw`。
- 無追蹤碼、無 Cookie，對學術與原住民社群較友善。

## 7. 內容治理（Editorial Policy）

- **引用來源**：所有語料必須標明出處（如 Paul Jen-kuei Li, Ogawa, 費羅禮等）。
- **標音系統**：建議以「教育部臺灣原住民族語言書寫系統」為主，搭配 IPA。
- **拼寫不確定之處**：以 `⟨…⟩` 包覆並加註「待審訂」，避免散佈錯誤。
- **尊重族人**：與地方部落或文化協會合作校訂。

## 8. 下一步行動（Handoff）

1. 將本 repo push 到 GitHub（建議組織帳號 `basay-tw` 或 `ketagalan`）。
2. 在倉庫 Settings → Pages 啟用 Pages，Source 選 `main` / `root`。
3. DNS：將 `basay.tw` 的 A record 指向 GitHub Pages IP，或 CNAME 指向 `<user>.github.io`。
4. 首批詞條與語法樣本由專家審訂後，替換 `data/dictionary.json` 與 `grammar/index.html` 的佔位內容。
