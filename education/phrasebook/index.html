母音系統的特殊性
巴賽語 Mutravai 方言共有六個母音：/i e a o u ə/。其中 /i/ 的 F2 異常偏低（約 1448 Hz），是該方言的真實特徵，在語音錄音與歌謠錄音中均可確認。以下共振峰數據來自 Asai Erin 1936 年田野調查錄音的聲學分析（iterative k-means v5 分類器）。

✍️ 正書法規則
巴賽語書寫系統的規則、TTS 前處理、兩聲線的差異

① 撇號輔音（Apostrophe Consonants）

原則
撇號（'）は基礎輔音字母に後置し、発音部位・方式が変化した音素を表す修飾符号。撇號は独立した文字ではなく、前の輔音と一体の音素単位を形成する。
n'
/ŋ/ 軟顎鼻音
s'
/ʃ/ 硬顎擦音
t'
/tʃ/ 塞擦音
r'
/rʲ/ 硬顎化流音

② 焦點詞綴（Focus Affixes）の書寫

插入詞綴 -um-（主事焦點 AV）
-um- は詞根の第一母音の直前に挿入する。連字符なし。
ali → umali
來（AV）
lakaw → lumakaw
走（AV）
kan → makan
吃（ma-AV）
後綴 -in（受事焦點 PV）・-an（處所焦點 LV）
詞根の直後に連接。母音衝突が生じる場合も連字符なし。
kan + -in → kanin
被吃（PV）
inum + -an → inuman
喝的場所（LV）
beka + -an → bekaan
被問的事（LV）

③ 重疊形（Reduplication）

進行貌・強調の重疊
詞根の第一音節（CV 或いは CVC）を複製して動詞の前に付加する。連字符なし。
la-lakaw
走走（進行）
lalaleona
跑（進行 AV）
ka-kanta
唱唱（強調）
※ lalaleona は la-（重疊/進行）+ um-（AV）+ leona（詞根「跑」）の複合形。Asai Erin 1936年手稿より復元（*lalaleona として推定）。

⑤ 兩聲線的音韻差異

bsy（IPay）と bsystd（台語適合）の違い
同一の正書法テキストを入力しても、二つの声線では異なる音響出力を生成する。bsy は Asai Erin 田野調査資料に基づく歴史的復元音韻を実装し、bsystd は現代の台湾語（閩南語）話者が習得しやすい音韻に Lobanov 正規化した標準声線。
🔵 bsy（IPay 聲線）
  • /i/ の F2 が低い（~1448 Hz）— Mutravai 方言の実測音価
  • 小舌音 /q/ を保持
  • 母音の弱化（ə）が顕著
  • 歴史的子音群の保持
  • 学術・史料的用途向け
🟢 bsystd（台語適合聲線）
  • Lobanov 正規化による標準化母音
  • /q/ → /k/ への近似
  • 台語話者の調音パターンに合わせた F1/F2
  • 閩南語リズムに近いプロソディ
  • 教育・普及用途向け

⑥ 大文字化・句讀點

文頭・固有名詞（人名・地名）は大文字で始める。句点は使わずピリオド（.）、疑問文（?）、感嘆文（!）を使用。撇號（'）は輔音修飾のみに使用し、他の目的（アポストロフィー、省略符号）には使用しない。
